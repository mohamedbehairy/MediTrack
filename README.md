# 🏥 MediTrack - Graduation Project (ACU 2026)

MediTrack is a full-stack healthcare system that helps manage the interaction between patients, doctors, and pharmacies.

It allows doctors to create and manage prescriptions, patients to receive medication reminders and track their doses, and pharmacies to manage stock and dispense medications efficiently.

The system improves medication adherence and provides better communication across all healthcare roles.
